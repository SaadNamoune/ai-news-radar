### [ReasonGate: Explainable Security Gate for LLM Prompt Injection](https://github.com/cgrtml/reasongate)

<small>**Hacker News - Newest: "llm"** &nbsp;·&nbsp; 2026-07-16 23:21:31 &nbsp;·&nbsp; `prompt-injection` `LLM-security` `explainable-ai` `security-gate`</small>

![ReasonGate: Explainable Security Gate for LLM Prompt Injection](https://repository-images.githubusercontent.com/1253754548/de48dd7e-e3aa-410b-a52e-05dd3d80519d)

**Overview:** ReasonGate is an explainable security gate for LLM applications that blocks prompt injection attacks, including indirect and zero-width-obfuscated variants. It addresses OWASP's top LLM security risk by providing auditable, model-agnostic defense with detailed reasoning for every decision. **Method:** The system inspects three surfaces: user prompt, retrieved context, and model output, using layered detectors (normalization, injection, ML-based semantic analysis, leakage, and provenance). A policy engine fuses signals via calibrated noisy-OR. The core is pure Python with zero dependencies, while enterprise add-ons provide embedding-based ML and provenance detectors. **Results:** Benchmarks on deepset/prompt-injections, jailbreak-classification, and safe-guard-prompt-injection datasets show 0.88 F1 on out-of-distribution data (vs. 0.97 in-distribution), with ablations revealing data artifacts in synthetic training. **Impact:** Advances explainable LLM security by providing transparent, auditable defenses for production deployments, addressing critical gaps in current black-box approaches.

### [Tracing PyTorch Autograd for Physics-Informed Neural Networks with Two-Level Differentiation](https://arxiv.org/abs/2607.13042)

<small>**cs.LG updates on arXiv.org** &nbsp;·&nbsp; 2026-07-16 05:00:00 &nbsp;·&nbsp; `automatic-differentiation` `physics-informed-neural-networks` `PyTorch-autograd`</small>

**Overview:** This paper provides a meticulous, node-by-node tracing of PyTorch’s automatic differentiation (AD) engine during Physics-Informed Neural Network (PINN) training, which requires two nested levels of differentiation: computing physics derivatives (e.g., $\hat{y}'(t)$) and parameter gradients ($
\nabla_\theta L$) of a loss dependent on those derivatives. **Method:** Using a 1-3-3-1 MLP and the ODE $y'(t)+y(t)=0$, $y(0)=1$, the authors trace the forward-pass computational graph, reverse-mode backward traversal (computing 22 parameter gradients in one pass), and the graph-on-graph mechanism enabled by `create_graph=True` for differentiating through physics-informed residuals. Every adjoint value is validated against hand-derived results from Tahimi (2026). **Results:** The paper explicitly quantifies gradient computations at each node, confirming correctness via cross-validation with analytical derivatives. **Impact:** Advances understanding of AD in PINNs, bridging sensitivity frameworks ($P/Q$) with PyTorch’s vector–Jacobian products, and provides a template for debugging or optimizing nested differentiation pipelines in scientific ML.

### [Symmetry Invisibility in Machine Learning: Null Fibers and Stabilizers via Lie Group Actions](https://arxiv.org/abs/2607.13046)

<small>**cs.LG updates on arXiv.org** &nbsp;·&nbsp; 2026-07-16 05:00:00 &nbsp;·&nbsp; `Lie-groups` `symmetry-invariance` `model-analysis` `quantum-circuits`</small>

**Overview:** Develops a mathematical framework to quantify symmetry information discarded by ML models when inputs carry a Lie group action, applicable to both classical neural networks and variational quantum circuits. **Method:** Defines the null fiber $N_G(f,x)$ (group elements undetectable by $f$) and stabilizer $\mathrm{Stab}_G(f)$ for functions $f:V\to\mathbb{R}$. Uses the preimage theorem to show null fibers have dimension $\geq \dim G - 1$ generically, and the Peter–Weyl theorem for spectral characterization under compact groups. Computes null fiber elements via Newton iteration on orbit maps. **Results:** Demonstrates applications in data masking, model fingerprinting, and privacy-preserving computation, with experiments on $\mathrm{SO}(3)$ molecular property prediction and $\mathrm{PSL}(2,\mathbb{C})$ spherical image classification. **Impact:** Advances theoretical understanding of symmetry in ML, enabling new tools for model analysis, robustness, and privacy. Opens avenues for symmetry-aware architecture design and certification.

**Overview:** ob introduces record- and token-level data provenance to address the critical gap in model unlearning: locating training records belonging to specific authors for removal. This matters because existing provenance systems operate at file/dataset level, causing catastrophic over-deletion. **Method:** ob propagates author identity through data processing pipelines and resolves revocation requests into precise forget sets via deterministic queries. For a 1.7B model, provenance-based forget sets improve unlearning by 42% over random baselines. **Impact:** Advances responsible AI by enabling precise, auditable model unlearning, reducing wasteful data deletion and improving compliance with data removal requests.
