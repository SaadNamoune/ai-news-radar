### [Semidirect Fourier Delta Attention: Phase-Controlled Linear Attention with Exact Chunk Transfer](https://arxiv.org/abs/2607.11897)

<small>**cs.LG updates on arXiv.org** &nbsp;·&nbsp; 2026-07-15 05:00:00 &nbsp;·&nbsp; `linear-attention` `Fourier-analysis` `recurrent-neural-networks` `state-space-models`</small>

**Overview:** This paper introduces Semidirect Fourier Delta Attention (SFDA), a phase-controlled generalization of linear attention that addresses the memory and state-tracking limitations of recurrent linear attention mechanisms. It provides exact affine chunk transfer and formal stability guarantees, enabling long-context memory. **Method:** SFDA replaces real diagonal decay in Kimi Delta Attention with block-rotational Fourier control, defined by the recurrence: S_t = (I - β_t k_t k_t^*)Λ_t S_{t-1} + β_t k_t v_t^*, where Λ_t = diag(α_t ⊙ e^{iθ_t}). The paper derives a constructive chunk-WY factorization for products of matrices A_t = Λ_t - u_t r_t^*, yielding exact affine chunk transfer with bounded rank growth. **Results:** Numerical verification confirms the algebra, and toy state-tracking experiments show SFDA learns cyclic memory where the phase-disabled KDA baseline fails. **Impact:** Advances linear attention mechanisms by providing exact state tracking and long-context memory, with potential applications in large-scale sequence modeling and state-space models.

### [Security-Oriented Metamorphic Testing and Association Rule Mining for LLM-Generated Code Vulnerabilities](https://arxiv.org/abs/2607.12089)

<small>**cs.CR updates on arXiv.org** &nbsp;·&nbsp; 2026-07-15 05:00:00 &nbsp;·&nbsp; `llm-security` `code-vulnerability-detection` `metamorphic-testing` `association-rule-mining`</small>

**Overview:** This paper introduces a framework combining Metamorphic Relations (MRs) and Association Rule (AR) mining to detect and analyze security vulnerabilities in LLM-generated code. The work addresses the cross-cutting nature of security flaws, which often span multiple categories simultaneously. **Method:** The framework defines nine MRs covering major CWE categories (e.g., SQL injection, XSS, command injection) and applies them via an LLM-based judge to 3,700 code snippets from five open models. AR mining is used to uncover co-violation patterns among vulnerabilities. **Results:** 68.8% of snippets violate at least one MR, with hard-coded credentials (79.1%) and command injection (74.4%) being highly prevalent. AR mining reveals strong co-violation patterns, such as XSS and weak cryptography predicting hard-coded credentials with 82.5% confidence (lift = 3.23). Prompt-level analysis shows database- and authentication-related prompts are strong predictors of insecurity. **Impact:** Demonstrates that insecure code generation is a structured, prompt-conditioned phenomenon, motivating cluster-aware verification and prompt-level interventions for safer LLM-assisted programming.

### [HARP-ME: GPU Framework for Exact Induced Motif Enumeration via Anchor-Reuse Partitioning](https://arxiv.org/abs/2607.12074)

<small>**cs.DC updates on arXiv.org** &nbsp;·&nbsp; 2026-07-15 05:00:00 &nbsp;·&nbsp; `graph-mining` `GPU-computing` `motif-enumeration` `graph-algorithms`</small>

**Overview:** HARP-ME is a GPU framework for exact enumeration of connected induced 4-node motifs in large graphs, addressing challenges like irregular candidate expansion and repeated set intersections. It advances graph mining by optimizing GPU execution for motif enumeration, a fundamental operation in network analysis. **Method:** HARP-ME introduces *closure-aware compilation* to select anchors for enumeration based on traversal cost, algebraic derivation benefits, and halo overhead, and *induced-signature reuse* to identify reusable completion states using candidate-frontier information and compact constraints. For out-of-memory graphs, it uses a canonical anchor-owner rule to ensure exact counting across partitions. **Results:** On six social, web, biological, and synthetic graphs, HARP-ME achieves up to 2.11x speedup over Pangolin, 1.83x over partitioned PBE, and 10.73x over a CPU baseline. It achieves 64-76% cache-hit rates and reduces host-to-device data transfer overhead compared to PBE-style partitioning. **Impact:** Demonstrates that optimizing anchor selection and algebraic derivation can complement traversal and reuse techniques, enabling scalable and efficient motif enumeration on GPUs.

**Overview:** OmniPM-Net is a hybrid model combining graph neural networks (GNNs) and chemical transport models (CTMs) to improve PM10 (particulate matter) forecasting accuracy and spatial resolution. It addresses the limitations of GNNs (station-scale accuracy but no gridded outputs) and CTMs (gridded forecasts with local biases) by fusing both into a unified framework. **Method:** The model uses a terrain-aware Gaussian set convolution to lift irregular GNN station forecasts onto a regular grid, followed by a multi-scale Spatial Source Attention (SSA) module to blend these with CAMS forecasts. It achieves a 9% reduction in 90th-percentile MAE over the GNN and 25% over CAMS, with notable gains during dust episodes. **Impact:** Advances air-quality forecasting by providing both high-accuracy station-level and gridded predictions, enabling better public health responses and environmental monitoring.

### [PhiCalNet: Removing Shape-Prior Shortcuts in Single-Shot Fringe Projection Profilometry via Architectural Constraints](https://arxiv.org/abs/2607.11928)

<small>**cs.LG updates on arXiv.org** &nbsp;·&nbsp; 2026-07-15 05:00:00 &nbsp;·&nbsp; `fringe-projection` `depth-estimation` `physics-informed-neural-networks` `uncertainty-quantification`</small>

**Overview:** PhiCalNet addresses a critical failure mode in single-shot fringe projection profilometry (FPP) networks, where models exploit shape-prior shortcuts (e.g., inferring depth from object boundaries) rather than fringe phase. This paper demonstrates that architectural changes, not just loss penalties, are necessary to eliminate such shortcuts. **Method:** PhiCalNet outputs a wrapped-phase representation (sinφ, cosφ) and maps it to depth via a fixed differentiable calibration layer, removing the shape-prior solution space. It takes fringe order as auxiliary input, and a sensitivity analysis shows robustness to realistic decoding errors. **Results:** On a photorealistic synthetic benchmark (15,600 images), PhiCalNet reduces object MAE 3.3x to 4.46 mm, with residuals confined to 0.103% of pixels at ±π wrap discontinuities. A three-frame extension achieves 1.16 mm MAE. Interpretability analysis confirms phase as the most decodable internal feature, and conformal uncertainty quantification localizes errors at discontinuities, cutting RMSE by 64% when rejecting the top 5% of pixels. **Impact:** Advances FPP by providing accurate depth estimation without shortcuts, with applications in 3D scanning, robotics, and industrial inspection.
