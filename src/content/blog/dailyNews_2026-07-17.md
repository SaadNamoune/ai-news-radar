### [SOUNDGATE: Enforcing Barrier Semantics in LLM-Agent Frameworks](https://arxiv.org/abs/2607.14166)

<small>**cs.SE updates on arXiv.org** &nbsp;·&nbsp; 2026-07-17 05:00:00 &nbsp;·&nbsp; `LLM-agents` `security` `side-effect-control` `formal-verification`</small>

**Overview:** This paper exposes critical **barrier semantics violations** in six widely used open-source LLM-agent frameworks, where control primitives (approval gates, cancellations, timeouts) fail to prevent side effects during pauses. The authors propose **SOUNDGATE**, an environment-external effect gate in Rust, to enforce strict hold-until-decided, reject-cancels, and dedup-on-replay semantics. **Method:** SOUNDGATE uses differential probes to isolate sibling leaks and other violations, then enforces properties via kernel-enforced routes and a Rust admission core. Verification includes model-checking (Verus, TLA+/TLC to 7.5e7 states) and Loom-based testing. **Results:** Blocks all measured violations end-to-end on all six frameworks while enabling legitimate effects, with ~1 ms admission per write and 12k-26k durable admissions per second. Frontier models trigger leaks at pooled rates up to 14%, and 215 of 1,200 runs execute side effects during pauses. **Impact:** Addresses a critical security and correctness gap in LLM-agent frameworks, providing a verifiable solution for side-effect control in production environments.

**Overview:** This paper introduces the Distributed Truncated Spectral Transform (DTST) to address inefficiencies in distributed Fourier Neural Operators (FNOs) for high-resolution computational physics. **Method:** DTST reverses the order of operations in distributed FFTs by computing only locally relevant frequency modes via partial DFTs, reducing communication payloads. ### [Lower Bound on Byzantine Agreement in Full-Information Adaptive Adversary Model](https://arxiv.org/abs/2607.14413)

<small>**cs.DC updates on arXiv.org** &nbsp;·&nbsp; 2026-07-17 05:00:00 &nbsp;·&nbsp; `byzantine-agreement` `distributed-algorithms` `lower-bounds`</small>

**Overview:** This paper proves a new lower bound on the worst-case expected round complexity of randomized synchronous Byzantine Agreement protocols in the full-information, strongly adaptive adversary model. **Method:** The proof introduces a multi-round concentration lemma showing that any transcript event of probability *p* can be forced by corrupting *O*(√(n log(1/*p*))) parties in expectation. It combines tools from prior work (e.g., Chor et al. 1989) to derive the bound Ω(t²/(n log(n+1))), matching recent upper bounds up to a log²n factor. **Results:** The bound improves upon the seminal Ω(t/√(n log n)) result of Bar-Joseph and Ben-Or (1998) and aligns with the O(min{t² log n / n, t / log n}) upper bound of Dufoulon and Pandurangan (2025). **Impact:** Advances the theoretical foundations of Byzantine fault tolerance, providing tighter complexity bounds for secure distributed protocols under adaptive adversaries.

**Overview:** This presentation proposes a method to distill frontier LLM capabilities into smaller, local SLMs by instrumenting AI-powered LSPs with OpenTelemetry. The approach leverages user interaction telemetry (accept/dismiss/regenerate actions) as implicit labels to continuously improve cheaper models. **Method:** Custom AI-LSPs extend standard rule-based checkers by embedding OpenTelemetry to capture fine-grained user feedback during coding sessions. This creates a feedback loop where accepted suggestions train local SLMs, reducing token costs while preserving semantic understanding. **Results:** Demonstrates practical implementation of proactive, agentic LSPs that integrate with VS Code/Neovim via LSP protocol, showing how compile-time feedback loops and observability tools (Criterion, flamegraphs) optimize concurrent code paths. **Impact:** Advances AI-assisted development by bridging frontier models with production-ready tooling, enabling scalable, cost-effective semantic code intelligence without sacrificing developer ergonomics.
